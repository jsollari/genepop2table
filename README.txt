This distribution includes material for the genepop2table computer program

The directory structure of this archive is as follows
+-base directory
	+--doc
	+--examples
	+--src

The base directory contains the following files:

README.txt  - this file
LICENSE.txt - GNU license v3 for the popABC software
Makefile - makefile to compile genepop2table executables

The contents of the subdirectories are as follows:

doc - this directory contains user guide for genepop2table

examples - this directory contains example files to use with genepop2table and to go through the quick-start guides

src - this directory contains the source code of the popABC package
